\section*{Введение}


В работе представлен численный алгоритм по определению диэлектрической проницаемости внутри области по модулю функции напряжения рассеянного электрического поля, измеренного на границе этой области. 
Это так называемая бесфазная обратная задача. В такой постановке используется только модуль комплекснозначной функции, отвечающей рассеянному на неоднородностях среды физическому полю, в данном случае электрическому. 
Задачи подобного типа возникают из физических экспериментов, проводящихся при высоких уровнях энергии (высоких частотах) для определения свойств среды, имеющей мелкие включения типа наноструктур. 
При таких физических условиях фазовая составляющая поля не может быть измерена, можно говорить только об измерении модуля рассеянного поля. 

Впервые подобная постановка бесфазной обратной задачи по определению потенциала в уравнении Шредингера по модулю рассеянного поля была поставлена в работе Шадана и Сабатье \cite{ShS}, так же о практической важности решения бесфазных обратных задач было указано Р.~Ньютоном \cite{New}. Первые результаты по исследованию единственности восстановления фазы по модулю амплитуды рассеяния были получены в работах М.В.~Клибанова \cite{Kli1}-\cite{Kli9}.

Различные конструктивные методы построения решения бесфазных обратных задач были предложены  в работах М.В.~Клибанова, В.Г.~Романова \cite{KR1}-\cite{KR3} и Р.Г.~Новикова \cite{N1,N2} для уравнения Шредингера,  М.В.~Клибанова и В.Г.~Романова \cite{KR4}-\cite{KR7} для обобщенного уравнения Гельмгольца, в работах В.Г.~Романова для уравнений Максвелла \cite{R1,R2}. 

Необходимо отметить, что в работах \cite{K+1}-\cite{K+6} по численому решению бесфазных обратных задач применяется и развивается предложенный М.В.~Клибановым глобальносходящийся метод (см., например, \cite{K+1}, \cite{BK1}-\cite{BK3}). Данный метод показал хорошую работу при решении различных задач. 

В данной работе мы применили оптимизационный метод решения обратной задачи.  

\section{Постановка задачи}

Рассмотрим стационарную систему уравнений Максвелла
\begin{equation}
{\rm rot} H= -i\omega\varepsilon(x)E+\tilde{j}, \ \ \ \ {\rm rot} E = i\omega\mu_{0}H, \label{901}
\end{equation}
в которой $\mu_{0}$ -- магнитная проницаемость, $\varepsilon(x)$ -- диэлектрическая проницаемость, $\tilde{j}=j^{0}\delta(x-y)$ -- точечный диполь, локализованный в точке $y$, и $j^{0}$ -- единичный вектор, определяющий направление диполя. Векторы $E$ и $H$ характеризуют напряженности электрического и магнитного полей соответственно. Здесь точка $y$ и частота $\omega$ являются параметрами задачи.

Относительно диэлектрической проницаемости предполагается, что она представляет собой гладкую положительную функцию всюду в $\mathbb{R}^{3}$, совпадающей с некоторой заданной положительной постоянной $\varepsilon_{0}$ вне некоторой компактной области $\Omega$, а носитель разности $\varepsilon(x)-\varepsilon_{0}$ строго лежит внутри $\Omega$, т.е. предполагаем, что существует некоторая окрестность границы $\partial\Omega$ области $\Omega$, где $\varepsilon(x)=\varepsilon_{0}$. 

Через $E^{sc}$ будем обозначать напряжение рассеянного электрического поля.\medskip

{\bf Задача}: Пусть во всех точках границы $\partial\Omega$ области $\Omega$ задана приведенная величина модуля напряжения электрического поля $\mathcal{E}(x,\omega,y)=\omega^{-2}|E^{sc}(x,\omega,y)|$, $(x,y)\in\partial\Omega\times\partial\Omega$, $\omega\ge\omega_{0}>0$. Необходимо определить $\varepsilon(x)$ внутри области $\Omega$. 

Эта задача рассматривалась в работе \cite{R1}, в которой доказана единственность ее решения.

\section{Вычисление $\mathcal{E}(x,\omega,y)$ при известной функции $\varepsilon(x)$}

В работах \cite{R1,R2} получена асимптотика 
\begin{equation}
\mathcal{E}=(A-B)^{2}+2AB\sin^{2}\!\bigl(\omega(\tau-\tau_{0})\bigr)+O(1/\omega), \label{001}
\end{equation}
$$
A=\frac{1}{2\tau}\,\bigl|a-\nu(a\!\cdot\!\nu)\bigl|, \ \ \ \ \ B=\frac{\mu_{0}n_{0}}{4\pi\tau_{0}}\,\bigl|j^{0}-\nu^{0}(j^{0}\!\cdot\!\nu^{0})\bigr|.
$$
Здесь обозначено
$$
n(x)=\sqrt{\mu_{0}\varepsilon(x)}, \ \ \ n_{0}=\sqrt{\mu_{0}\varepsilon_{0}}, \ \ \ \nu(x,y)=\frac{\nabla\tau(x,y)}{n(x)},\ \ \ \nu^{0}(x,y)=\frac{x-y}{|x-y|}, \ \ \ \tau^{0}=n_{0}|x-y|,
$$
$\tau(x,y)$ и $\tau_{0}(x,y)$ -- времена пробега электромагнитной волны в неоднородной и однородной среде, вектор-функция $a=a(x,y)$ является решением дифференциального уравнения
\begin{equation}
4\bigl(\nabla\tau\cdot\nabla\bigr)a+\bigl(-6n^{2}(x)+\Delta\tau^{2}\bigr)a+2\tau\bigl(\nabla\tau\cdot a\bigl)\nabla\ln\epsilon=0 \label{003}
\end{equation}
с краевыми условиями $a|_{s=0}=(\mu_{0}n_{0}/2\pi)j^{0}$.

При получении соотношения \eqref{001} предполагалось, что Риманова метрика $d\tau=n(x)|dx|$ является простой, т.е. считалось, что любые две точки $x$, $y$ из $\mathbb{R}^3$ могут быть соединены единственной геодезической $\Gamma(x,y)$.

Таким образом, для того, чтобы вычислить $\mathcal{M}(x,\omega,y)$ необходимо уметь вычислять величины $\tau(x,y)$, $\Delta\tau(x,y)$ и вектора $\nabla\tau(x,y)$, $a(x,y)$.

Для вычисления величины $\tau(x,y)$ и вектора $\nabla\tau(x,y)$ существует хорошо известный путь (см., например, \cite{BV}) -- это решение уравнения эйконала

\begin{equation}
|\nabla\tau|^{2}=n^{2}, \label{004}
\end{equation}
которое сводится к решению следующей системы уравнений:
\begin{eqnarray}
\dot{x} &=& p/n^{2}, \ \ \  x|_{s=0}=y, \label{005}\\[2mm]
\dot{p} &=& \nabla\ln n, \ \ \ p|_{s=0}=n(y)\eta^{0},\label{006}
\end{eqnarray}
где введены стандартные обозначения $p_{k}=\partial\tau/\partial x_{k}$ ($k=\overline{1,3}$), точка означает дифференцирование по паратметру $s$, и $\eta^{0}$ -- единичный вектор, указывающий направление входа луча в область $\Omega$. Если считать, что имеет место начальное условие $\tau|_{s=0}=0$, то имеем равенство $s=\tau(x,y)$.

Введем обозначение $q_{km}=\partial p_{k}/\partial x_{m}$ ($k,m=\overline{1,3}$). Очевидно, имеет место равенство
\begin{equation}
q_{km}=q_{mk}. \label{007}
\end{equation}

Из уравнения \eqref{004} следует 
\begin{equation}
\sum_{j=1}^{3}p_{j}\frac{\partial p_{k}}{\partial x_{j}}=n\,\frac{\partial n}{\partial x_{k}}. \label{008}
\end{equation}
Дифференцируем равенство \eqref{008} по $x_{m}$, получим
$$
\sum_{j=1}^{3}p_{j}\frac{\partial^{2} p_{k}}{\partial x_{j}\partial x_{m}}+\sum_{j=1}^{3}\frac{\partial p_{j}}{\partial x_{m}}\,\frac{\partial p_{k}}{\partial x_{j}} 
=n\,\frac{\partial^{2} n}{\partial x_{k}\partial x_{m}}+\frac{\partial n}{\partial x_{k}}\,\frac{\partial n}{\partial x_{m}},
$$
откуда, учитывая введенные обозначения, равенства \eqref{007} и определение дифференцирования по параметру $s$, получаем
\begin{equation}
\dot{q}_{km}+\frac{1}{n^{2}}\sum_{j=1}^{3}q_{kj}q_{jm}=\frac{1}{2n^{2}}\bigl(n^{2}\bigr)_{x_{k}x_{m}}. \label{009}
\end{equation}

Краевым условием для нахождения $q_{km}$ ($k,m=\overline{1,3}$) может служить соотношение
\begin{equation}
q_{km}|_{x\to y} \sim n_{0}\left(\delta_{km}-\frac{x_{k}-y_{k}}{|x-y|}\,\frac{x_{m}-y_{m}}{|x-y|}\right)\frac{1}{|x-y|}  \label{010}
\end{equation}
($\delta_{km}$ -- дельта Кронекера), которое следует из предположения, что существует некоторая окрестность границы $\partial\Omega$, где $\tau(x,y) \sim n_{0}|x-y|$.

Введем матрицы $Q$, $N$ и $\mathcal{V}$ с элементами $Q_{km}=q_{km}$, $N_{km}=\frac{1}{2}n^{-2}\bigl(n^{2}\bigr)_{x_{k}x_{m}}$ и $\mathcal{V}_{km}=\eta_{k}^{0}\eta_{m}^{0}$.
Данные матрицы являются симметричными.

Уравнения \eqref{009} и начальные условия \eqref{010} можно записать в компактной форме:
\begin{equation}
\dot{Q}+\frac{1}{n^{2}}Q^{2}=N, \ \ \ \ \ Q|_{x\to y} \sim \frac{n_{0}}{|x-y|}\bigl(E-\mathcal{V}\bigr).  \label{011}
\end{equation}

Учитывая введенные обозначения, решения уравнений \eqref{005}-\eqref{006}, определение дифференцирования по параметру $s$, равенства $\nabla\ln\varepsilon=\nabla\ln n^{2}$ и $\Delta\tau^{2}=2|\nabla\tau|^{2}+2\tau\Delta\tau=2n^{2}+2\tau\Delta\tau$, можем переписать дифференциальное уравнение \eqref{003} в следующем виде:
\begin{equation}
4n^{2}\,\dot{a}+\bigl(-4n^{2}+2\tau\Delta\tau\bigr)a+2\tau\bigl(p\cdot a\bigl)\nabla\ln n^{2}=0. \label{003a}
\end{equation}
Нетрудно видеть, что $\Delta\tau={\rm tr\,}Q$.

Объединим соотношения \eqref{005}, \eqref{006}, \eqref{011}, \eqref{003a} в одну систему уравнений. Однако краевое условие в \eqref{011} является неудобным для реализации какого-либо численного метода решения. 
Воспользуемся тем, что в окрестности границы $\partial\Omega$ величина $n(x)$ постоянная, $n(x)=n_{0}$. Найдем решение объединенной системы для некоторого $s=s_{0}$. 

Прежде всего, легко показать, что $\mathcal{V}^{2}=\mathcal{V}$. Поскольку в окрестности границы $\partial\Omega$ матрица $N=0$, тогда матрица $Q$ есть решение матричного уравнения $\dot{Q}+n^{-2}_{0}Q^{2}=0$ и 
$Q=\bigl(n_{0}^{2}/s\bigr)\!\bigl(E-\mathcal{V}\bigr)$, следовательно, $Q|_{s=s_{0}}=\bigl(n_{0}^{2}/s_{0}\bigr)\!\bigl(E-\mathcal{V}\bigr)$.

Из \eqref{006} следует, что $p|_{s=s_{0}}=n_{0}\eta^{0}$. Отсюда и из \eqref{005} получаем $x|_{s=s_{0}}=y+(s_{0}/n_{0})\eta^{0}$. 

Для матрицы $\bigl(n_{0}^{2}/s\bigr)\!\bigl(E-\mathcal{V}\bigr)$ верно равенство ${\rm tr\,}Q=2n^{2}_{0}/s$, следовательно, для некоторой окрестности границы $\partial\Omega$ второй член равенства \eqref{003a} равен нулю, третий член этого равенства равен нулю в силу постоянства $n(x)$. Следовательно, вектор-функция $a$ удовлетворяет дифференциальному уравнению $\dot{a}=0$, отсюда и краевого условия из \eqref{003} следует $a|_{s=s_{0}}=(\mu_{0}n_{0}/2\pi)j^{0}$.

Вышеприведенные рассуждения позволяют написать объединённую систему уравнений в удобном виде для численного решения
\begin{eqnarray}
\dot{x} &=& p/n^{2}, \ \ \ \ \  x|_{s=s_{0}}=y+(s_{0}/n_{0})\eta^{0}, \label{016} \\[2mm]
\dot{p} &=& \nabla\ln n, \ \ \ \ \ p|_{s=s_{0}}=n_{0}\eta^{0}, \label{017}\\[2mm]
\dot{Q} &=& -\frac{1}{n^{2}}Q^{2}+N, \ \ \ \ \ Q|_{s=s_{0}}=\frac{n_{0}^{2}}{s_{0}}\bigl(E-\mathcal{V}\bigr), \label{018} \\[2mm]
\dot{a} &=& \left(1-\frac{s\!\cdot\!{\rm tr\,}Q}{2n^{2}}\right)a-s\bigl(p\cdot a\bigl)\frac{\nabla\ln n^{2}}{2n^{2}}, \ \ \ \ \  a|_{s=s_{0}}=\frac{\mu_{0}n_{0}}{2\pi}j^{0}. \label{019}
\end{eqnarray}

Таким образом, решая систему \eqref{016}-\eqref{019} для некоторого известного $n(x)$, для каждой точки $y$ и направления $\eta^{0}$ в ней, для каждого направления диполя $j^{0}$ можно вычислить точку $x$ на $\partial\Omega$, времена пробега $\tau(x,y)$ между точками $y$ и $x$ и вектор-функцию $a(x,y)$. Зная эти значения, можно вычислить значение приведенного модуля электромагнитного поля $\mathcal{E}(x,\omega,y)$ (см. \eqref{001}).
 
Для численного решения системы \eqref{016}-\eqref{019} использовался метод Рунге-Кутта четвертого порядка, для тестирования данного решения был реализован метод Адамса 5-ого и 13-ого порядков по схеме предиктор-корректор (см., например, \cite{Bah,Ort,Hol}). 

\section{Численный алгоритм решения обратной задачи}

Численный метод решения обратной задачи восстановления параметра диэлектрической проницаемости основан на сведении обратной задачи модуля $\mathcal{E}(x_k,\omega,y_l)$ к обратной кинематической задаче. Для этого, на первом этапе решения обратной задачи по характеристикам периода функции $\mathcal{E}(x_k,\omega,y_l)$ вычисляется время пробега $\tau(x,y)$ (вычисляются данные обратной кинематической задачи), на втором этапе решается обратная кинематическая задача, входными параметрами которой являются вычисленные времена пробега.


Без потери общности можно считать, что область $\Omega$ представляет собой куб со стороной $l$, а расстояние от его границы до границы носителя больше некоторой константы $l_0>0$. Причем, при необходимости эту константу можно сделать сколь угодно большой за счет увеличения размеров куба.

Как было показано в работе \cite{R1}, рассматриваемая обратная задача сводится к обратной кинематической задаче, поэтому можно считать, что в качестве исходных данных для обратной задачи нам уже заданы времена пробега $\tau(x,y)$ между произвольными точками границы области $\Omega$. Кроме того, в численных вычислениях нам будет удобно вместе с коэффициентом диэлектрической проницаемости $\varepsilon(x)$ использовать индекс рефракции $n(x)$, связанный с $\varepsilon(x)$ соотношением $n(x)=\sqrt{\varepsilon(x)\mu_0}$. В однородной среде он так же постоянен $n_0=\sqrt{\varepsilon_0\mu_0}$.

Как было сказано ранее, можно считать, что область представляет собой куб со стороной $l$. Для дискретизации задачи разобьем данный куб на $n^3$ элементарных кубов (\ref{cube}), в которых коэффициент диэлектрической проницаемости и индекс рефракции -- постоянны и равны $\varepsilon_i$ и $n_i$.

\begin{figure}[h]
\centering
\includegraphics{img//cube.ps}
\caption{Дискретная модель обратной задачи.}
\label{cube}
\end{figure}

Будем считать, что точки $x_k$ и $y_l$ находятся в центрах квадратов, выходящих на границу исходного куба (тем самым задана некоторая регулярность в данных). Таким образом, в дискретном виде обратная задача может быть переформулирована следующим образом: по данным $\mathcal{E}_*(x_k,\omega,y_l)$ или $\tau_*(x_k, y_l)$, измеренных в точках на границе куба, найти $\varepsilon_i$ и $n_i$ внутри.

Отметим, что исходные данные обратной задачи представляют собой достаточно большой массив данных. Если исходная область была разбита на $n^3$ элементарных кубов, то множество точек $x_k$ состоит из $6n^2$ элементов. Соответственно, множество данных  $\mathcal{E}_*(x_k,\omega,y_l)$ (для фиксированной частоты) или $\tau(x_k, y_l)$ состоит из $15n^4$ элементов. Этим же дополнительно обосновывается необходимость перехода от значений функции $\mathcal{E}_*(x_k,\omega,y_l)$ как функции частоты к функции $\tau(x_k, y_l)$ (объем данных уменьшается пропорционально количеству измерений модуля рассеянного поля как функции частоты).

Численно обратную задачу будем решать минимизируя функционал невязки:

\begin{equation}
F=\sum (\tau(x_k, y_l) - \tau_*(x_k, y_l))^2.
\label{resudual_func}
\end{equation}
Здесь суммирование ведется по всем парам точек на границе куба, а величина $\tau(x_k, y_l)$ представляет собой данные для расчетной задачи.

\subsection{Численный алгоритм нахождения периода почти периодической функции}

Алгоритм вычисления периода основан на использовании данных о последовательных минимумах и максимумах функции $\mathcal{E}(x,\omega,y)$. Пусть задан конечный набор частот $\omega^{min}_i$ ($i=1,2,\ldots,N$), $\omega^{max}_j$ ($j=1,2,\ldots,M$), на которых функция достигает своих минимумов и максимумов. Тогда, принимая во внимание, что разность между последовательными значениями $\omega^{min}_i(\tau(x,y)-\tau_0(x,y))$ и  $\omega^{max}_i(\tau(x,y)-\tau_0(x,y))$ должна стремиться к периоду $\pi$ функции $\sin^2(t)$, получим следующие выражения:
\begin{eqnarray}
\omega^{min}_N(\tau(x,y)-\tau_0(x,y))-\omega^{min}_{N-1}(\tau(x,y)-\tau_0(x,y)) \approx \pi, \\
\omega^{max}_M(\tau(x,y)-\tau_0(x,y))-\omega^{max}_{M-1}(\tau(x,y)-\tau_0(x,y)) \approx \pi.
\end{eqnarray}

Указанные выражения дают 2 значения $\tau^{min}(x,y)$ и $\tau^{max}(x,y)$. Определим $\tau(x,y)$ по следующей формуле:
\begin{equation}
\tau(x,y)=\frac{\tau^{min}(x,y) + \tau^{max}(x,y)}{2}.
\end{equation}
Данное значение будет использоваться при решении обратной кинематической задачи.

Стоит сделать следующее замечание. Очевидным образом, при решении обратной задачи для реальных данных результаты измерений доступны для конечного диапазона частот $\omega$. Если на этом диапазоне частот минимумов и максимумов измеряемой функции меньше двух, то вычисление $\tau(x,y)$ становится проблематичным. В случае ограниченности данных можно воспользоваться полупериодом -- разницей между частотами, соответствующим максимуму и минимуму. Но при этом, для обеспечения точности решения задачи, эти экстремумы должны быть вычислены с достаточной степенью достоверности. 




\subsection{Свойства функционала невязки}

Для того, чтобы эффективно решать обратную задачу, необходимо понимать, какими свойствами обладает функционал невязки, какую он имеет форму. 

В связи с тем, что визуализировать функционал, зависящий от $n^3$ параметров достаточно непросто, выберем некоторые однопараметрические семейства данных, на которых можно будет выявить основные свойства функционала невязки.

Рассмотрим внутри куба сферическую область, в которой индекс рефракции постоянен. Для простоты, будем считать, что в остальной части куба он так же постоянен и равен $1$. Таким образом, получим обратную задачу по восстановлению одного параметра. Функционал невязки в этом случае представляет собой функцию одной переменной.

Рассмотрим 4 разных ситуации когда индекс рефракции внутри этой сферы равен $1.1$, $1.2$, $1.3$ и $1.4$. На рис. \ref{image1} изображены графики функционала невязки. Из этого рисунка видно, что несмотря на присутствие локальных минимумов, функционал имеет явно выраженный глобальный минимум, соответствующий точному решению обратной задачи.
